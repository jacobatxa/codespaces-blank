行业打标全链路架构

StarRocks（人力数据）→ Python 预处理 → Agent + Skill → LLM → 飞书表格

Python 脚本

  • 从 SR 查询原始数据（公司名/经营范围）
  • 关键词提取：局/银行/科技/智能 等
  • 数据清洗、格式化、缓存
  • 生成 LLM 可用的结构化输入
  • 纯数据处理，不需要智能判断

LLM (DeepSeek)

  • 接收预处理特征 + 公司信息
  • 结合 Skill 注入的行业标准
  • 推理判断 → 一级/二级行业
  • 处理模糊、歧义、罕见公司名
  • 输出结构化打标结果

Agent 编排

  • 决定何时 SQL 查 SR、何时调 Python
  • 加载合适的 Skill 分类标准
  • 特征+标准→构造 Prompt→调 LLM
  • 结果→调用 lark-cli 写回
  • 循环批处理
