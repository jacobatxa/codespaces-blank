🧠 大模型与基础架构

Transformer 架构之外，新架构探索方兴未艾。状态空间模型（Mamba）、线性注意力机制等创新尝试不断涌现，试图突破传统 Transformer 在长序列推理上的二次方复杂度瓶颈。但正如刘壮所言，数据质量和训练策略对模型性能的影响可能远大于架构选择，行业正在重新审视「规模定律」的边界条件。

🤯 AI Agent与智能体

AI Agent 正从实验走向生产级部署。MCP（Model Context Protocol）和 A2A（Agent-to-Agent）两大协议标准的推进，正在为智能体互操作性铺平道路。在软件开发领域，以 OpenHands、Cursor 和 Claude Code 为代表的 AI Coding Agent 已经嵌入真实开发流程，从辅助工具升级为核心生产力。与此同时，AutoGPT 等通用自主 Agent 框架持续迭代，长时任务执行和多工具编排能力显著提升。百度 GenFlow 4.0 更将 Agent 能力扩展到 Office 三件套——文档生成、表格分析和幻灯片制作全自动化，实现「十分钟干完一周活」。

🤖 具身智能与机器人

具身智能的技术栈正在快速收敛。感知层面，多模态融合（视觉+触觉+力觉）成为标配；决策层面，基于大模型的技能学习替代传统的强化学习手工设计奖励函数；执行层面，灵巧手和力矩传感的成熟使精细操作成为可能。政策层面，中国「十四五」规划将具身智能列为重点发展方向，多个省市出台专项扶持政策，产业链上游的传感器、电机、减速器等核心零部件国产化进程加速。

🎨 多模态与AIGC

多模态能力正在从实验室走向产品。CLIP 和 SAM 等视觉基础模型被广泛集成到各类应用中，从图像搜索、自动标注到医学影像分析。语音交互方面，Whisper 的本地部署方案使得语音转写不再依赖云端 API，在隐私敏感场景中具有独特价值。端侧多模态模型的出现进一步拓展了应用场景——手机拍照实时翻译、AR 眼镜的环境理解、智能家居的视觉感知，多模态 AI 正在从专业工具变为日常基础设施。

💾 AI硬件与芯片

AI 芯片竞争从云端延伸到端侧。NVIDIA 凭借 CUDA 生态持续主导训练市场，但推理侧格局正在变化——Groq 的 LPU 架构在推理速度上实现数量级突破，AMD 的 ROCm 生态加速追赶。中国市场方面，昇腾芯片在政企和运营商场景落地加速，已进入金融、电力等关键行业的核心业务系统。寒武纪和地平线聚焦自动驾驶和边缘计算赛道，形成差异化竞争。端侧 AI 部署成为最热新趋势：高通骁龙 8 Gen 4 和联发科天玑系列均集成专用 NPU，使得 7B 级别模型可以在手机上跑推理。
